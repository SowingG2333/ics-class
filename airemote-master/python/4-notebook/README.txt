远程推理应用示例

远程推理将算法应用与模型推理分离，分别运行于Atlas200DK开发板和PC通用Python环境，
并支持分布式运行于集群平台中，全时在线、随处接入，可更好地支持教学实践，以及课外
创新应用实践。

本示例所展现的主要特点有：

1. 调用模型算法的远程推理接口，并编码实现从输入获取数据、后续处理及展示等其他流程；
2. 使用 Inferemote.DataSource 包，内置支持多种模态数据输入，包括图片目录、视频
   文件、摄像头或者图片的HTTP超链接地址；
3. 使用 Inferemote.InferenceJob 包进行远程推理，内置支持多线程；
4. 使用 Inferemote.tesing 包，支持结果的多种形式展示，包括图像窗口和WEB页面动态
   显示。

运行环境

本示例代码运行与 Ubuntu 20.04 的 Jupyter(Python3.9) 环境，仅可展示部分功能，
其他内容请在Python原生环境下使用。模型服务运行于远程的Atlas200DK开发板，不在本
示例中体现。

远程推理平台

本示例与所使用的远程推理集群服务器均运行在华为云ECS服务器上，集群中的12台Atlas200DK
开发板部署于电子科技大学校园.
